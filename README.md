# Purps Crypto

Eine Kryptowährungs-Management-Anwendung.

## Installation

```bash
npm install
```

## Starten

```bash
node server.js
```

Server läuft auf http://localhost:3000

## Dateien

- `server.js` - Express Server
- `database.js` - SQLite Datenbank
- `public/` - Frontend HTML/CSS/JS
- `sessions/` - Session-Daten
